### Shader
+ Android不仅可以使用颜色来填充图形，也可以使用Shader对象指定的渲染效果来填充图片；Shader本身是一个抽象类，它提供了如下实现类：

|实现类|说明
|------|-------|
|BitmapShader|使用位图平铺的渲染效果|
|LinearGradient|使用线性渐变来填充图形|
|RadialGradient|使用圆形渐变来填充图形|
|SweepGradient|使用角度渐变来填充图形|
|ComposeShader|使用组合渲染效果来填充图形|
